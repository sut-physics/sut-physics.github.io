ระหว่าง tab Dashboard และ เปรียบเทียบ Sheet ผมอยากเพิ่ม  tab "รายละเอียดงบประมาณ"  โดยจะเก็บที่ google sheet แบบนี้ ตอนนี้มีหลักๆคือ งบดำเนินงาน(แบ่งเป็นค่าใช้สอย ค่าวัสดุ ค่าจ้าง ค่าเดินทางไปต่างประเทศ อาจจะแบ่งได้อีก) งบลงทุน(ครุภัณฑ์ อาจจะมีได้อีก) โดยตอนนี้ 
งบดำเนินงาน(A62) ค่าใช้สอย(A63) จะแบ่งเป็นรายการย่อยจาก A65-A80 มีงบประมาณที่ B65-B80 ใช้ไปแล้ว C65-C80 และเหลือ D65-D80 ของแต่ละรายการย่อย ส่วน รวม(A64) ค่าของรวมงบประมาณ(B64) ค่าของรวมใช้ไปแล้ว(C64) ค่าของรวมเหลือ(D64) 
งบดำเนินงาน(F62) ค่าวัสดุ(F63) จะแบ่งเป็นรายการย่อยจาก F65-F80 มีงบประมาณที่ G65-G80 ใช้ไปแล้ว H65-H80 และเหลือ I65-I80 ของแต่ละรายการย่อย ส่วน รวม(F64) ค่าของรวมงบประมาณ(G64) ค่าของรวมใช้ไปแล้ว(H64) ค่าของรวมเหลือ(I64)
งบดำเนินงาน(K62) ค่าจ้าง(K63) จะแบ่งเป็นรายการย่อยจาก K65-L80 มีงบประมาณที่ L65-L80 ใช้ไปแล้ว M65-M80 และเหลือ N65-N80 ของแต่ละรายการย่อย ส่วน รวม(K64) ค่าของรวมงบประมาณ(L64) ค่าของรวมใช้ไปแล้ว(M64) ค่าของรวมเหลือ(N64)
งบดำเนินงาน(P62) ค่าเดินทางไปต่างประเทศ(P63) จะแบ่งเป็นรายการย่อยจาก P65-P80 มีงบประมาณที่ Q65-Q80 ใช้ไปแล้ว R65-R80 และเหลือ S65-S80 ของแต่ละรายการย่อย ส่วน รวม(P64) ค่าของรวมงบประมาณ(Q64) ค่าของรวมใช้ไปแล้ว(R64) ค่าของรวมเหลือ(S64)  
งบลงทุน(U62) ครุภัณฑ์(U63) จะแบ่งเป็นรายการย่อยจาก U65-U80 มีงบประมาณที่ V65-V80 ใช้ไปแล้ว W65-W80 และเหลือ X65-X80 ของแต่ละรายการย่อย ส่วน รวม(U64) ค่าของรวมงบประมาณ(V64) ค่าของรวมใช้ไปแล้ว(W64) ค่าของรวมเหลือ(X64)  
บางช่องอาจจะยังไม่มีค่าอะไรเลยเพราะทำเผื่อไว้จะมีเวลา ADD เพิ่มใน WEB 
โดยในหน้า TAB รายละเอียดงบประมาณ อยากให้แสดงข้อมูลที่มีเป็นตาราง มีปุ่ม "แก้ไข" "เพิ่มรายการ" โดยต้องเลือกแก้ไขไปที่ตอนนี้ที่มีคือ งบดำเนินงาน งบลงทุน และหน่อยย่อยของมันคือ อะไรเช่น ค่าใช้สอย ค่าวัสดุ ค่าจ้าง ค่าเดินทางไปต่างประเทศ ครุภัณฑ์ จะได้ชัดเจนว่าแก้ งบไหน ค่าอะไรในงบนั้น ปุ่มเพิ่มก็เหมือนกันมีให้เลือกงบว่างบไหน ค่าอะไร หรือ งบใหม่ ค่าใหม่ ที่ไม่มีในรายการ พิมพ์ได้ ถ้าเป็นรายการใหม่ที่มีในงบและค่าในนั้นแล้วก็เอาไปต่ออันเดิมที่มี ถ้ายังก็ต้องสร้างมาใหม่เพื่อบันทึก
